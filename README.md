# Bank Marketing Neural Network

Neural Network implementation from scratch using NumPy for binary classification on the Bank Marketing dataset.

## Goals

- Data preprocessing pipeline
- Exploratory Data Analysis
- Neural Network from scratch
- Backpropagation implementation
- Model evaluation
- Experiment tracking

## Dataset

Bank Marketing Dataset from UCI Machine Learning Repository.

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn
- Git
